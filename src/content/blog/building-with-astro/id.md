Astro telah sepenuhnya mengubah cara saya berpikir tentang membangun situs web statis. Beralih dari kerangka kerja sisi klien yang berat untuk situs berbasis konten adalah sebuah pencerahan.

### Arsitektur Pulau Komponen

Arsitektur pulau milik Astro berarti saya hanya mengirimkan JavaScript di tempat yang mutlak diperlukan. Situs ini, misalnya, hampir sepenuhnya berupa HTML dan CSS statis. Satu-satunya JavaScript yang berjalan di klien adalah logika untuk navigasi perangkat seluler dan formulir kontak.
