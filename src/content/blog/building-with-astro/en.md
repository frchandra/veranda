Astro has completely changed how I think about building static websites. Moving away from heavy client-side frameworks for content-driven sites was a revelation.

### The Component Island Architecture

Astro's islands architecture means I only ship JavaScript where it's absolutely necessary. This site, for example, is almost entirely static HTML and CSS. The only JavaScript running on the client side is for the mobile navigation and the contact form logic.
