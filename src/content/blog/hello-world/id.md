### Mengapa Astro?

Saya memilih [Astro](https://astro.build/) karena selaras dengan tujuan saya untuk menghasilkan situs statis, yang membutuhkan nol JavaScript secara bawaan, dan sangat cocok untuk blog yang berfokus pada teks seperti ini.

Berikut adalah cuplikan kode singkat untuk menguji penyorotan sintaksis Shiki:

```javascript
// Ucapkan halo pada dunia
function sapa(nama) {
  console.log(`Halo, ${nama}! Selamat datang di Veranda.`);
}

sapa("Pengembara");
```
