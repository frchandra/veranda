### なぜAstroなのか？

私が[Astro](https://astro.build/)を選んだ理由は、静的ファーストの出力という私の目標に合致しているからです。デフォルトでJavaScriptを必要とせず、このようにテキスト中心のブログに最適です。

Shikiの構文のハイライトをテストするための簡単なコードスニペットはこちらです：

```javascript
// 世界に挨拶する
function greet(name) {
  console.log(`こんにちは、${name}！Verandaへようこそ。`);
}

greet("旅人");
```
